在 Mac 上计算你写了多少行代码：

- 支持检测参与计算的代码文件夹数量以及代码行数；
- 支持同时检测多文件 / 多文件夹代码；
- 支持自定义检测的文件类型，例如: C，Swift，OC，Java... 类型的代码文件；
- 支持自定义需要忽略检测的文件夹, 例如: iOS 工程中的 Pods 文件夹；
- 支持中文与英文，跟随系统语言变化 (v1.1.0 起支持)；
- 支持忽略代码中的空行，代码行数计算更精确(v1.2.0 起支持)。
